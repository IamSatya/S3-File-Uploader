Troubleshooting:
----------------
If background doesn't appear on a page:
- Check browser console for image loading errors
- Verify hacktivate-bg.jpg exists in attached_assets folder
- Rebuild: npm run build
- Clear browser cache completely

If old background code remains:
- Make sure HacktivateBackgroundLayout.tsx is in client/src/components/
- Check that all page files are updated to import and use the layout
- Rebuild and restart PM2
