HackTIvate REBRANDING - INSTALLATION GUIDE
==========================================

This package contains the complete rebranding from "HackFiles" to "HackTIvate" 
with the beautiful HackTIvate 2025 background image.

Files Included:
---------------
1. hacktivate-bg.jpg     → attached_assets/hacktivate-bg.jpg (background image)
2. index.html            → client/index.html (page title & meta)
3. Landing.tsx           → client/src/pages/Landing.tsx (login page with background)
4. Dashboard.tsx         → client/src/pages/Dashboard.tsx (dashboard header)
5. WelcomeInstructions.tsx → client/src/components/WelcomeInstructions.tsx (help section)

Installation Steps:
-------------------
1. Download all files from attached_assets folder

2. Copy files to your local project:
   hacktivate-bg.jpg     → /var/www/hackfiles/attached_assets/hacktivate-bg.jpg
   index.html            → /var/www/hackfiles/client/index.html
   Landing.tsx           → /var/www/hackfiles/client/src/pages/Landing.tsx
   Dashboard.tsx         → /var/www/hackfiles/client/src/pages/Dashboard.tsx
   WelcomeInstructions.tsx → /var/www/hackfiles/client/src/components/WelcomeInstructions.tsx

3. Build and restart:
   cd /var/www/hackfiles
   npm run build
   pm2 restart all

4. Clear browser cache:
   - Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
   - Or open in Incognito mode

Troubleshooting:
----------------
If you see old "HackFiles" branding:
- Clear your browser cache completely
- Try opening in Incognito/Private window
- Make sure you ran npm run build before pm2 restart

If background image doesn't show:
- Verify hacktivate-bg.jpg is in attached_assets folder
- Check browser console for 404 errors
- Rebuild: npm run build
